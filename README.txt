Pseudo spectral method for Fractional norm gradient descent of Modica-Mortola Energy or Cahn-Hilliard energy.

Written by Kangping Zhu in OPENCL
kangping@cims.nyu.edu

Derived from Brian Wetton's algorithm and matlab code, wetton@math.ubc.cd

Targeting AMD Turks GPU for Initial Version

Change in ch2d.c for different hardware

To compile, using make. While the executable takes in 3 variables as s for H^s soblev norm, epsilon for microsopic length scale and N for numbers of data point in one dimension.

Code is in working progress
